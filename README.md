# AutoRace_2023

Written based on ROBOTIS_autorace_2020 

## Requirments
  1. Ubuntu 20.04
  2. ROS-Noetic
  3. OpenCV-4.5.0
  4. Tutrtlebot3 Setup
  5. ELP_Webcam (170 degree)
